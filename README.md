# Installation Termux || طريقة التثبيت تيرماكس

```bash
> pkg update && pkg upgrade
> pkg install -y nodejs
> pkg install -y ffmpeg
> pkg install -y git
> git clone https://github.com/adhkarbot/bot-termux
> cd bot-termux
> npm i
> node index.js
```
